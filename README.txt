Итоговая работа по курсу "Python-разработчик" по проекту "Содействие занятности"

# Описание работы
Программа получает фотографии профиля с ВК (ввод ID с консоли). Для этого используется токен VK и метод photos.get
Прежде, чем сохранять фото на Я.Диске, используются вспомогательные методы для получения фотографий максимального размера (ширина, высота в пикселях)
Также при парсинге для имени используется количество лайков на фотографии
Для подключения к Яндекс.Облаку используется токен.
При загрузке фотографии пользователь выбирает название для папки, которая будет создана для загрузки фотографий
При загрузке отображается прогрес-бар с помощью сторонней библиотеки tqdm
Конечный результат доступен по ссылке https://disk.yandex.ru/client/disk/From_VK_to_YandexCloud
Дополнительно информация по фотографиям сохранятся в json-файле в проекте

В будущем проект может быть улучшен (например, более детальный парсинг по различным параметрам, поиск по альбомам,
загрузка на другие облачные хранилища, добавление пользовательского интерфейса)
